# Security Policy

## Supported Versions

Currently, only the latest version of ISLI is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of ISLI seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1.  **Do not** open a public GitHub issue for security reports.
2.  Send an email to **itp.coder@gmail.com** with the details of the vulnerability.
3.  Include a detailed description of the issue, steps to reproduce, and the potential impact.

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix if applicable. We ask that you maintain confidentiality until we have had a chance to address the issue.
